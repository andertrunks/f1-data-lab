# F1 Data Lab

Projeto pessoal de análise de dados de Fórmula 1, criado como parte da minha transição profissional para a área de Dados, BI, SQL, Python e Automação.

## Objetivo

Construir uma base de dados simples sobre Fórmula 1 e usar essa base para praticar:

- organização de dados;
- criação de tabelas;
- consultas SQL;
- análise exploratória;
- dashboards em Power BI;
- documentação de projeto no GitHub.

## Primeira versão do projeto

Nesta primeira etapa, o projeto terá uma base pequena com:

- pilotos;
- equipes;
- corridas;
- temporadas;
- resultados.

## Perguntas que o projeto pretende responder

1. Quais pilotos têm mais vitórias?
2. Quais equipes têm mais vitórias?
3. Como as vitórias evoluíram por temporada?
4. Quais pilotos se destacaram em determinadas épocas?

## Ferramentas previstas

- CSV para organizar os dados;
- SQL para consultas;
- Power BI para dashboards;
- Python futuramente para automação e análise;
- GitHub para documentação e portfólio.
